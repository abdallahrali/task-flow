# 📝 TaskFlow - PHP To-Do List Application 

A simple, secure, and dynamic to-do list application built with PHP and MySQL. This project allows users to register, log in, and manage their personal tasks through a clean and responsive web interface. It's designed as a practical example of fundamental web development concepts, including user authentication, database interactions with PDO, and dynamic page generation.

## ✨ Key Features 

* **User Authentication**: Secure user registration and login system with password hashing.
* **CRUD Functionality**: Create, Read, Update, and Delete tasks.
* **Task Management**: Mark tasks as complete or pending with a single click.
* **Dynamic Filtering**: Filter tasks to view "All," "Pending," or "Completed" items.
* **User-Specific Tasks**: Each user can only view and manage their own tasks.
* **Responsive Design**: A clean, mobile-friendly interface built with Bootstrap.
* **Secure Code**: Uses prepared statements (PDO) to prevent SQL injection attacks.

## 🛠️ Tech Stack 

* **Backend**: PHP
* **Database**: MySQL
* **Frontend**: HTML, CSS, JavaScript
* **Frameworks/Libraries**:
    * [Bootstrap 5]
* **Development Environment**: XAMPP (Apache, MySQL, PHP)
